# Project Section for CV: J.A.R.V.I.S. (AI Assistant)

### **Project Name: J.A.R.V.I.S. (Just A Rather Very Intelligent System)**

**Overview:**
Developed a sophisticated hybrid AI personal assistant capable of voice-activated system automation, hardware monitoring, and natural language interaction. Features a robust triple-layer brain architecture combining cloud-based and local LLMs for high availability and privacy.

**Technical Stack:**
*   **Languages:** Python
*   **AI/Machine Learning:** Google Gemini (Flash & Pro), Ollama (Local LLM: Llama 3.2), Natural Language Intent Parsing.
*   **Voice Interaction:** SpeechRecognition (Google API), pyttsx3 (Text-to-Speech), SoundDevice, PyAudio.
*   **UI/UX:** Tkinter (Custom Futuristic Interface), Real-time Audio Visualizers.
*   **Automation:** PyAutoGUI (Human-like UI automation), psutil (Hardware telemetry), PowerShell Integration.
*   **Architecture:** Multi-threaded execution, JSON-based Command Protocol, Cloud/Local Hybrid Infrastructure.

**Key Contributions & Features:**
*   **Hybrid LLM Integration:** Engineered a fail-safe intelligence layer that switches between Google Gemini (Cloud) and Ollama/Llama 3.2 (Local) based on connectivity and latency.
*   **Natural Language Automation:** Developed a custom intent parser that translates complex voice commands into structured JSON actions for system-level automation (launching applications, automated typing, and file management).
*   **Real-time Visualization:** Designed and implemented a futuristic, high-performance GUI using Tkinter, featuring a dynamic ARC visualizer that reacts to system load and audio input.
*   **Concurrency Management:** Leveraged Python’s threading module to handle simultaneous audio capturing, TTS output, and UI updates without blocking the main execution thread.
*   **System Telemetry:** Integrated real-time hardware monitoring for CPU, RAM, and battery status, providing voice-activated system diagnostics.
